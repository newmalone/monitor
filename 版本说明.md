# 版本说明

## 版本号
v1.0.0

## 版本概述
完成设备状态监控系统首个可用版本，具备设备监控、历史查询、数据比对、地图展示、Excel 导入、局域网共享访问等核心能力。

## 版本功能
- 设备状态监控
- 文件上传与自动解析
- 历史快照留存与调取
- 双日期数据比对
- 高德地图设备点位展示
- 后端统一数据存储
- 后台管理风格前端界面

## 本版本重点修复
- 修复上传字段过长导致的 MulterError
- 修复旧上传链路引起的 `"undefined" is not valid JSON`
- 调整为服务端直接解析 Excel
- 优化前端错误提示

## 技术栈
- React 18
- TypeScript 5
- Vite 5
- Ant Design 5
- Express 4
- XLSX 0.18

## 运行方式
- 前端：`npm run dev`
- 后端：`npm run server`

## 局域网访问
- 前端默认监听 `0.0.0.0:3000`
- 后端默认监听 `0.0.0.0:3001`
