# 设备监控 AGENT 自定义报表功能设计文档

## 一、功能概述

本功能通过自然语言对话的方式，让用户与 AGENT 交互，快速获取设备状态信息和生成自定义报表。

---

## 二、常用查询与报表类型

### 2.1 简单统计查询类

| 查询场景 | 示例输入 | 查询逻辑 | 输出格式 |
|---------|---------|---------|---------|
| 今日在线设备数 | "今天在线设备有多少" | 获取最新快照，筛选启用状态且在线的设备 | 数字 + 简单说明 |
| 今日离线设备数 | "今天有多少离线设备" | 获取最新快照，筛选启用状态且离线的设备 | 数字 + 简单说明 |
| 今日异常设备数 | "有多少设备异常" | 获取最新快照，筛选启用状态且异常的设备 | 数字 + 简单说明 |
| 在线率查询 | "今天的在线率是多少" | 计算在线设备 / 总启用设备 | 百分比 + 简要分析 |
| 指定日期统计 | "2026-06-02 的在线设备数" | 获取指定日期快照进行统计 | 数字 + 简单说明 |

### 2.2 对比分析查询类

| 查询场景 | 示例输入 | 查询逻辑 | 输出格式 |
|---------|---------|---------|---------|
| 两日在线率对比 | "昨天和今天的在线率对比" | 获取两天快照，计算在线率并对比 | 表格 + 趋势分析 |
| 设备数变化对比 | "这两天在线设备变化多少" | 获取两天快照，统计差值 | 数字 + 变化描述 |
| 月度趋势分析 | "6月份的在线率变化趋势" | 获取6月份所有快照，计算每日在线率 | 折线图 + 趋势描述 |

### 2.3 自定义报表生成类

| 查询场景 | 示例输入 | 查询逻辑 | 输出格式 |
|---------|---------|---------|---------|
| 生成日报 | "生成6月2日的设备状态日报" | 获取指定日期数据，生成标准日报 | 日报预览 + 导出按钮 |
| 生成周报 | "生成上周的设备周报" | 获取一周数据，汇总统计 | 周报预览 + 导出按钮 |
| 生成对比报表 | "生成昨天和今天的对比报表" | 获取两天数据，生成对比报表 | 对比报表预览 + 导出按钮 |

### 2.4 异常设备排查类

| 查询场景 | 示例输入 | 查询逻辑 | 输出格式 |
|---------|---------|---------|---------|
| 离线设备列表 | "列出所有离线设备" | 获取最新快照，筛选离线设备 | 表格（IP、位置、类型等） |
| 长期离线设备 | "列出离线超过3天的设备" | 获取最近几天快照，对比状态 | 表格 + 离线时长 |
| 新增离线设备 | "今天新离线的设备有哪些" | 对比昨日和今日快照，找出新增离线 | 表格 + 状态变化 |
| 按类型筛选异常 | "毫米波雷达有哪些异常" | 按设备类型筛选异常设备 | 表格 + 统计数据 |
| 按厂家筛选 | "列出A厂家所有离线设备" | 按厂家名称筛选离线设备 | 表格 + 统计数据 |

### 2.5 多轮对话示例

```
用户：在线设备有多少？
AGENT：今天在线设备有 1976 台，在线率 96.58%

用户：其中属于华为的有多少？
AGENT：华为的在线设备有 456 台

用户：帮我列出来
AGENT：[华为在线设备表格]
```

#### 3.2.2 API 设计

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/agent/chat` | POST | 发送消息并获取回复 |
| `/api/agent/conversations` | GET | 获取对话历史列表 |
| `/api/agent/conversations/:id` | GET | 获取指定对话详情 |
| `/api/agent/conversations/:id` | DELETE | 删除指定对话 |
| `/api/agent/export` | POST | 导出生成的报表 |

#### 3.2.3 请求/响应格式

**聊天请求：**
```json
{
  "conversationId": "可选，用于多轮对话",
  "message": "用户输入的自然语言",
  "context": {
    "currentPage": "monitor|compare|history|report",
    "selectedDate": "2026-06-02"
  }
}
```

**聊天响应：**
```json
{
  "conversationId": "conv_123",
  "messageId": "msg_456",
  "response": {
    "type": "text|table|chart|report",
    "content": "自然语言回复内容",
    "data": {
      // 对应类型的数据
    },
    "actions": [
      {
        "type": "export",
        "format": "word|pdf|excel",
        "label": "导出报表"
      }
    ]
  }

```

### 3.3 意图解析设计

#### 3.3.1 意图类型定义

```javascript
const INTENT_TYPES = {
  // 统计查询
  SIMPLE_STAT: 'simple_stat',        // 简单统计
  ONLINE_COUNT: 'online_count',      // 在线数
  OFFLINE_COUNT: 'offline_count',    // 离线数
  ABNORMAL_COUNT: 'abnormal_count',  // 异常数
  ONLINE_RATE: 'online_rate',        // 在线率
  
  // 对比分析
  COMPARE: 'compare',                // 对比查询
  TREND: 'trend',                    // 趋势分析
  
  // 报表生成
  GENERATE_REPORT: 'generate_report', // 生成报表
  DAILY_REPORT: 'daily_report',      // 日报
  WEEKLY_REPORT: 'weekly_report',    // 周报
  COMPARISON_REPORT: 'comparison_report', // 对比报表
  
  // 异常排查
  LIST_DEVICES: 'list_devices',      // 列出设备
  LONG_OFFLINE: 'long_offline',      // 长期离线
  NEW_OFFLINE: 'new_offline',        // 新增离线
  
  // 筛选查询
  FILTER_BY_TYPE: 'filter_by_type',  // 按类型筛选
  FILTER_BY_MANUFACTURER: 'filter_by_manufacturer', // 按厂家筛选
  FILTER_BY_REGION: 'filter_by_region', // 按区域筛选
  
  // 其他
  UNKNOWN: 'unknown'
};
```

#### 3.3.2 参数提取

需要从用户输入中提取的关键参数：
- `date`: 日期或日期范围
- `deviceType`: 设备类型
- `manufacturer`: 厂家名称
- `region`: 区域
- `status`: 状态（在线/离线/异常）
- `duration`: 时长（如"3天"）
